BlockChain é

O que é BlockChain?

É um sistema de rastreamento digital.

Como Funciona?

Pedaços de código são gerados e conectados como um bloco de informações.

1. Dados

Pedaços de código são gerados e conectados em uma corrente de blocos ordenada.

2. Ledger

O Ledger é composto por blocos de dados que são autenticados em tempo real para evitar fraude.

3. Criptografia e P2P

Todos os computadores de uma rede Peer to Peer guardam uma cópia do Ledger para autenticação compartilhada em tempo real.
